    Moby(Docker)

    Dockerе…¬еҸёзӣҙжҺҘжҠҠеҺҹDockerйЎ№зӣ®ж”№еҗҚжҲҗдәҶMobyпјҡhttps://github.com/moby/moby

    еҸӮиҖғд№ҰзұҚпјҡгҖҠDocker е®№еҷЁдёҺе®№еҷЁдә‘гҖӢпјҢгҖҠж·ұе…Ҙеү–жһҗKubenetesгҖӢ

    docker = image + cgroup + namespace

    • image и§ЈеҶізҡ„жҳҜжү“еҢ…й—®йўҳпјҢзӣёеҜ№дәҺдј з»ҹPaasпјҢдҝқиҜҒејҖеҸ‘/з”ҹдә§зҺҜеўғзҡ„дёҖиҮҙжҖ§пјӣ
    • е®Ңж•ҙзҡ„ж–Үд»¶зі»з»ҹпјҡrootfsпјҲchrootпјүпјӣ
    • иҮӘе®ҡд№үзҡ„жү“еҢ…жөҒзЁӢпјҡDockerfileпјӣ
    • cgroup е’Ң namespace жҸҗдҫӣиҝҗиЎҢзҡ„жІҷзӣ’пјҢжҳҜLinuxзҡ„жңәеҲ¶пјҢи·ҹдј з»ҹPaasдёҖиҮҙпјӣ

    еҹәзЎҖзҹҘиҜҶ

    • й•ңеғҸжҳҜеӨҡеұӮеӯҳеӮЁпјҢжҜҸдёҖеұӮзҡ„дёңиҘҝе№¶дёҚдјҡеңЁдёӢдёҖеұӮиў«еҲ йҷӨпјҢдјҡдёҖзӣҙи·ҹйҡҸзқҖй•ңеғҸпјӣ
    • е‘Ҫд»ӨиЎҢе·Ҙе…·dockerдёҺDocker daemonе»әз«ӢйҖҡдҝЎпјҢDocker daemonжҳҜDockerе®ҲжҠӨиҝӣзЁӢпјҢиҙҹиҙЈжҺҘж”¶е№¶еҲҶеҸ‘жү§иЎҢDockerе‘Ҫд»Өпјӣ
    • dockerе‘Ҫд»Өжү§иЎҢйңҖиҰҒиҺ·еҸ–rootжқғйҷҗпјҢз»‘е®ҡеұһдәҺrootзҡ„Unix Socketпјӣ

    • еҗҢдёҖеҸ°жңәеҷЁдёҠзҡ„жүҖжңүе®№еҷЁе…ұдә«дё»жңәж“ҚдҪңзі»з»ҹзҡ„еҶ…ж ёгҖӮ

    • еҰӮжһңеә”з”ЁзЁӢеәҸйңҖиҰҒй…ҚзҪ®еҶ…ж ёеҸӮж•°гҖҒеҠ иҪҪйўқеӨ–еҶ…ж ёжЁЎеқ—гҖҒдёҺеҶ…ж ёзӣҙжҺҘдәӨдә’пјҢжҳҜе…ЁеұҖеҸҜи§Ғзҡ„гҖӮ

    • dockerй•ңеғҸиҝҗиЎҢж—¶пјҢдјҡе…ҲеҲӣе»әдёҖдёӘе®№еҷЁеҲқе§ӢеҢ–иҝӣзЁӢпјҲdockerinitпјүпјҢе®ҢжҲҗж №зӣ®еҪ•еҮҶеӨҮгҖҒи®ҫеӨҮжҢӮиҪҪзӯүеҲқе§ӢеҢ–ж“ҚдҪңпјҢеңЁйҖҡиҝҮexecv()зі»з»ҹи°ғз”Ёи®©еә”з”ЁиҝӣзЁӢеҸ–д»ЈиҮӘе·ұжҲҗдёәе®№еҷЁPID=1зҡ„иҝӣзЁӢпјӣ

    й•ңеғҸж–Үд»¶

    DockerдҪҝз”ЁдәҶLinuxжң¬иә«зҡ„bootfsпјҢеҸӘйңҖиҰҒе®үиЈ…иҮӘе·ұжүҖйңҖзҡ„rootfsгҖӮ

    • е®№еҷЁVolumeйҮҢзҡ„дҝЎжҒҜдёҚдјҡиў«docker commitжҸҗдәӨпјҢдҪҶжҳҜжҢӮиҪҪзӮ№зӣ®еҪ•дјҡеҮәзҺ°еңЁж–°зҡ„й•ңеғҸдёӯгҖӮ

    • й•ңеғҸзҡ„еұӮж”ҫеңЁ/var/lib/docker/aufs/diffзӣ®еҪ•дёӢпјҢиў«иҒ”еҗҲжҢӮиҪҪеңЁ/var/lib/docker/aufs/mntдёӯгҖӮ

    иҒ”еҗҲжҢӮиҪҪ

    й•ңеғҸи®ҫи®ЎиҝҮзЁӢдёӯеј•е…ҘдәҶеұӮпјҲlayerпјүзҡ„жҰӮеҝөпјҢеҲ¶дҪңй•ңеғҸзҡ„жҜҸдёҖжӯҘйғҪдјҡз”ҹжҲҗдёҖдёӘеұӮпјҢеҪўжҲҗеўһйҮҸзҡ„rootfsпјӣ

    • UnionFSпјҡе°ҶдёҚеҗҢдҪҚзҪ®зҡ„зӣ®еҪ•иҒ”еҗҲжҢӮиҪҪпјҲunion mountпјүеҲ°еҗҢдёҖдёӘзӣ®еҪ•дёӢпјӣ

    иҜ»еҶҷеұӮ

    readwriteеұӮпјҢеҲ йҷӨfooж–Үд»¶дјҡйҖҡиҝҮж–°еўһ.wh.fooж–Үд»¶пјҢиҝӣиЎҢ"йҒ®жҢЎ"гҖӮ

    docker commitе’ҢpushжҢҮд»Өдҝқеӯҳдҝ®ж”№иҝҮзҡ„еҸҜиҜ»еҶҷеұӮпјҢеҪўжҲҗж–°зҡ„й•ңеғҸпјҢе…ұеҲ«дәәдҪҝз”ЁгҖӮ

    иө„жәҗйҷҗеҲ¶

    CPU/Memroy еңЁе®№еҷЁеҶ…жҳҜзңӢеҲ°йҷҗеҲ¶зҡ„пјҢиҝҳжҳҜе®ҝдё»жңәзҡ„жүҖжңүдҝЎжҒҜпјҹ

    еҹәдәҺcgroupйҷҗеҲ¶

    • Docker daemon дјҡеңЁеҚ•зӢ¬жҢӮиҪҪжҜҸдёҖдёӘеӯҗзі»з»ҹзҡ„жҺ§еҲ¶з»„зӣ®еҪ•пјҲеҰӮ/sys/fs/cgroup/cpuпјүдёӢеҲӣе»әеҗҚдёәdockerзҡ„жҺ§еҲ¶з»„пјӣ
    • еңЁdockerжҺ§еҲ¶з»„йҮҢпјҢдёәжҜҸдёӘе®№еҷЁеҲӣе»әдёҖдёӘд»Ҙе®№еҷЁIDдёәеҗҚз§°зҡ„е®№еҷЁжҺ§еҲ¶з»„пјҢе®№еҷЁеҶ…зҡ„жүҖжңүиҝӣзЁӢеҸ·дјҡеҶҷеҲ°иҜҘжҺ§еҲ¶з»„tasksдёӯпјҢе№¶еңЁжҺ§еҲ¶ж–Үд»¶дёӯпјҲеҰӮcpu.cfs_quota_usеҶҷе…Ҙйў„и®ҫзҪ®зҡ„еҖјпјүпјӣ

    зЈҒзӣҳиө„жәҗйҷҗеҲ¶

    cgroupдёҚж”ҜжҢҒеҜ№зЈҒзӣҳиө„жәҗиҝӣиЎҢйҷҗеҲ¶пјҢиҖҢQuotaжҠҖжңҜеҹәдәҺз”ЁжҲ·е’Ңж–Үд»¶зі»з»ҹпјҢиҖҢдёҚжҳҜеҹәдәҺиҝӣзЁӢжҲ–иҖ…зӣ®еҪ•зЈҒзӣҳйҷҗйўқгҖӮ

    еҸҜиЎҢж–№жЎҲпјҡ

    1. жҜҸдёӘе®№еҷЁдёҖдёӘз”ЁжҲ·пјҢжүҖжңүз”ЁжҲ·е…ұдә«е®ҝдё»жңәдёҖеқ—зЈҒзӣҳпјҢд»…еҜ№жҷ®йҖҡз”ЁжҲ·жңүж•Ҳпјӣ
    2. йҖүжӢ©ж”ҜжҢҒеҜ№жҹҗдёҖдёӘзӣ®еҪ•еҸҜд»Ҙйҷҗйўқзҡ„ж–Үд»¶зі»з»ҹпјҢеҰӮXFSпјӣ
    3. еҲӣе»әиҷҡжӢҹж–Үд»¶зі»з»ҹпјҢжӯӨж–Үд»¶зі»з»ҹд»…дҫӣжҹҗдёҖдёӘе®№еҷЁдҪҝз”ЁгҖӮ

    е®№еҷЁжөҒйҮҸйҷҗеҲ¶

    Trafic Controller жҳҜLinuxзҡ„жөҒйҮҸжҺ§еҲ¶жЁЎеқ—пјҢеҺҹзҗҶжҳҜдёәж•°жҚ®еҢ…е»әз«ӢйҳҹеҲ—пјҢе№¶дё”е®ҡд№үйҳҹеҲ—дёӯж•°жҚ®еҢ…зҡ„еҸ‘йҖҒи§„еҲҷгҖӮ

    docker жҸҗдҫӣз»ҷжҲ‘д»¬еӨҡз§Қ(4з§Қ)зҪ‘з»ңжЁЎејҸпјҡ

    • ж— зҪ‘з»ңжЁЎејҸ пјҡnoneпјҢжңүзӢ¬з«Ӣзҡ„network nsпјҢдҪҶжҳҜйңҖиҰҒз”ЁжҲ·еҚ•зӢ¬й…ҚзҪ®пјӣ
    • е®ҝдё»зҪ‘з»ңжЁЎејҸ пјҡhostпјҢдёҚе…·еӨҮзӢ¬з«Ӣзҡ„network nsпјӣ
    • е®№еҷЁзҪ‘з»ңжЁЎејҸпјҡcontainerпјҢж–°е®№еҷЁдёҺжҹҗдёӘе·ІеӯҳеңЁзҡ„е®№еҷЁе…ұдә«network nsпјҢе®№еҷЁй—ҙеҸҜйҖҡиҝҮеӣһзҺҜзҪ‘еҚЎйҖҡдҝЎпјӣ

    BridgeжЁЎејҸ

    docker0 зҪ‘еҚЎ/зҪ‘жЎҘпјҡ

    • зҪ‘жЎҘзҡ„жҰӮеҝөзӯүеҗҢдәҺдәӨжҚўжңәпјҢдёәиҝһеңЁе…¶дёҠзҡ„и®ҫеӨҮиҪ¬еҸ‘ж•°жҚ®её§пјҢvethзӣёеҪ“дәҺз«ҜеҸЈпјҢе·ҘдҪңеңЁдәҢеұӮпјҢдёҚйңҖиҰҒIPдҝЎжҒҜпјӣ
    • docker0д№ҹжҳҜдәҢеұӮи®ҫеӨҮпјҢдҪҶдёәжҷ®йҖҡзҡ„LinuxзҪ‘жЎҘпјҢеҸҜд»Ҙй…ҚзҪ®IPпјҢдҪңдёәе®№еҷЁзҡ„й»ҳи®ӨзҪ‘е…іпјӣ

    Dockerе®үиЈ…еҘҪж—¶пјҢеңЁiptablesй…ҚзҪ®SNATи§„еҲҷпјҲжәҗең°еқҖиҪ¬жҚўпјүпјҢеҸҜд»Ҙи®ҝй—®еӨ–йғЁ
